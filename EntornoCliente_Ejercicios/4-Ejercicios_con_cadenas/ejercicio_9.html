Programa una función que dado un array de números devuelva un objeto con dos arrays, el primero contendrá el contenido del array original ordenado en forma ascendente y el segundo, el contenido del array original ordenado de forma descendiente.

Ej ascendenteDescendente(([3, 5,3,8,6]) devolverá { asc: [3,3,5,6,8], desc: [8,6,5,3,3] }.
